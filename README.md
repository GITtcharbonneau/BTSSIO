# Mon Projet Collab

Bienvenue dans notre projet collaboratif ! Ce projet propose une nouvelle fonctionnalité passionnante : la possibilité d'écrire en bleu.

## Fonctionnalité "Écrire en Bleu"

Pour utiliser cette fonctionnalité, suivez ces étapes simples :

1. Ajoutez la classe CSS `.bleu` aux éléments que vous souhaitez afficher en bleu.
   ```html
   <div class="bleu">
       <!-- Votre contenu en bleu ici -->
   </div>
